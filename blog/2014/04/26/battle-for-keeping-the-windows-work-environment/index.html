Battle for Keeping the Windows Work Environment

Apr 26th, 2014 | Comments

So I’m still fighting to improve my Windows environment. It seemed to be good enough until a couple of days ago, but then I got a new toy at work.

Some days ago I got a Mac Mini to use as my desktop at work and became addicted to OSX again. It’s not easy to escape from Apple environment once you experience it.

It’s not perfect, but I’ve to admit that it’s the closer to the ideal environment I’ve ever experienced. It has lots of little details which aren’t big things, but when you add all of them together they become something really enjoyable to work with.

The one thing which really is a must is the Apple keyboard. Once you get used to it you won’t want to go back to anything else. So much that I had to give up and order one for using at home. Never looked back to my other keyboards since then.

Next, the two most productive details for me are Spotlight and fast virtual desktop switching (don’t know exactly how it’s called is OSX). After I experienced they again on OSX at work I knew I had to get it working at home as well.

I went out shopping for solutions which could deliver these features for Windows and found FARR (Find and Run Robot) for acting as a Spotlight-like solution and VirtualWin, offering the fast virtual desktop switching solution.

Both of them are free applications which work really well together. They allow me to have separated virtual desktops for research, work, fun, learning, remote shell access, etc, as well as start applications really fast without needing to use the mouse.

Also, I’ve to point out that if you use glasses and need to keep the brightness of your screen under control at night, flu.x is your friend (it’s multi-platform, ie, it works under Windows, Linux and OSX).

Prior to these adjustments, I was considering buying a Mac Mini for home usage. Actually, it’s not somehting completely out of question yet. However, now I can have the most important features (for me at least) in a environment I already have at home.

Buying a Mac Mini would be the best option, but not something I would do right now as it’s a little bit expensive for me, even if the Mini is the cheapest way to run OSX, and specially if you’re a Brazilian.

The Mini would be an awesome product to have at home and everything I could need to run under Windows or Linux could be done using a virtual machine under the Mini.

Also, despite VirtualBox working fine under OSX, I suspect buying a Parallels license would be worth it as the features it offers when it comes to nicely integrate Windows and/or Linux applications inside OSX’s environment are very interesting.

Well, not today kid. Not today :–)
